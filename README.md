# Task Todo System

Project ini adalah aplikasi Task Management (To-Do System) berbasis web yang dibangun dengan:

-   Laravel 12
-   Filament Admin v4
-   PostgreSQL
-   Spatie Activitylog

## Installation

```bash
  git clone https://github.com/Mahib22/task-todo.git
  cd task-todo
  composer install
  npm install && npm run build
  cp .env.example .env
  php artisan key:generate
  php artisan migrate
  php artisan serve
```
